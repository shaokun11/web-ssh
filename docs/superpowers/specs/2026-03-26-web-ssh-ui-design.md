# Web SSH 界面设计规范

> 创建日期: 2026-03-26
> 状态: 待审核

## 1. 设计概述

### 1.1 设计目标
- 打造专业级的 Web SSH 终端界面
- 提供流畅的用户体验
- 支持多连接管理
- 本地数据存储，跨设备可用

### 1.2 设计风格
- **整体风格**: 暗黑专业风格（类似 Termius）
- **配色方案**: Dracula 主题
- **设计语言**: 现代、简洁、高对比度

## 4. 组件设计

### 4.1 Header 组件

```
┌─────────────────────────────────────────────────────────────┐
│ [Logo] [Status Dot] [Connection Info]    [Theme] [Settings] [Disconnect] │
└─────────────────────────────────────────────────────────────┘
```

- Logo: 品牌色 `#bd93f9`，16px 粗体
- 状态点: 在线 `#50fa7b`，离线 `#6272a4`，带发光效果
- 断开按钮: 危险色 `#ff5555` 背景

### 4.2 连接列表项

```
┌─────────────────────────────────┐
│ ● production-server             │  ← 选中: 紫色边框 + 半透明背景
│   user@192.168.1.100:22         │  ← 灰色小字
└─────────────────────────────────┘
```

状态:
- 已连接: 左边框 `#bd93f9` (紫色) + 绿色状态点
- 断开: 普通边框 `#44475a` + 灰色状态点

## 5. 交互设计

### 5.1 状态反馈

| 状态 | 视觉表现 |
|------|----------|
| 连接中 | 状态点闪烁动画 |
| 已连接 | 绿色状态点 + 发光效果 |
| 断开 | 灰色状态点 |
| 错误 | 红色背景提示条 |

### 5.2 动画效果

- 过渡时间: 150ms (快速响应)
- 状态点闪烁: 1s 循环
- 悬停效果: 背景色渐变
- 弹窗出现: 淡入 + 缩放

### 5.3 响应式设计

| 屏幕宽度 | 布局调整 |
|----------|----------|
| > 1200px | 完整三栏布局 |
| 768-1200px | 隐藏历史面板 |
| < 768px | 侧边栏抽屉化，显示虚拟键盘 |
